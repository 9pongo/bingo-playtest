# 90-Ball Video Bingo — Playtest (standalone)

A **playtest build** of the 90-ball, 4-card video bingo game — the full PixiJS
client with an **in-browser mock backend**, so it runs with no server. For
*feel / UX* evaluation, not for server-authority / certification (the real game
is server-authoritative).

- Live: https://9pongo.github.io/bingo-playtest/
- Play money starts at 100,000 COIN; jackpot seed 5,000 COIN (RTP up to 95.84%).
- The game math mirrors the server (the RNG byte-matches, so the built-in
  provably-fair check still verifies cards + the 30-ball main draw), but
  outcomes are computed client-side in this build.

Single self-contained bundle (`assets/index.js`) generated by
`client/build-standalone.mjs` in the main project.
